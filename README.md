# 🏨 Dashboard de Gestão de Alojamentos
### Unidade Curricular: Gestão de Alojamentos (ESGHT-UAlg)

Este projeto é uma ferramenta interativa desenvolvida para automatizar e facilitar o cálculo de métricas essenciais na gestão hoteleira.

---

## 🚀 Funcionalidades Principais

### 1. Fórmula de Hubbart (Bottom-Up)
Calcula a Diária Média (ADR) necessária para atingir os objetivos financeiros do investidor
**Input:** Investimento, ROE, IRC, Empréstimos e Custos Operacionais 
**Output:** Receita exigida e preço médio por quarto

### 2. KPIs & Markup Líquido
Análise em tempo real da saúde financeira do alojamento
Cálculo automático de **ADR, RevPAR e NetRevPAR**
Verificador de sustentabilidade baseado no **Markup Líquido**

### 3. Engenharia de Variáveis (Cálculos Inversos)
Módulo avançado para descobrir dados em falta, permitindo resolver diversos cenários:
**Descobrir o COPAR** através do RevPAR e Markup
**Descobrir o RevPAR Mínimo** através do COPAR e Markup pretendido
**Cálculo de COPAR** via Lucro Líquido e Volume de Vendas

### 4. Análise de Custos e Estratégia
**Break-Even Point:** Volume de vendas necessário para cobrir custos fixos e variáveis
**Regra 1€ por 1000€:** Estimativa rápida de preço baseada no investimento
**Simulador Hit or Miss:** Comparação de rentabilidade entre épocas baixa, média e alta

## 🎨 Design e Interface
**GitHub Dark Mode:** interface visual inspirada nas cores oficiais do GitHub para maior conforto ocular
**Gráficos Dinâmicos:** visualização da distribuição de receitas de lucro vs impostos vs custos
**Otimizado para Impressão:** relatórios formatados especificamente para gerar PDFs limpos e organizados
