# Calculadora - Fórmula de Hubbart 🏨

Uma calculadora web interativa para calcular a **Diária Média (ADR)** na gestão hoteleira utilizando o método *Bottom-Up* (Fórmula de Hubbart).

🪐 **[Clica aqui para testar a calculadora online](https://joquinhaexe.github.io/hubbartcalculadora/)** 🪐

## 📖 Sobre o Projeto
A Fórmula de Hubbart é uma técnica matemática essencial na hotelaria que em vez de nos basearmos no preço dos quartos na concorrência( o famoso cego que guia outros cegos), 
este método calcula o valor exato que deve ser cobrado por noite para cobrir todos os custos operacionais, impostos e garantir a margem de lucro (ROE) exigida pelos investidores.

Este projeto foi criado para simplificar e automatizar este cálculo permitindo visualizar em tempo real de forma a ver como as diferentes variáveis afetam o preço final.

## ✨ Funcionalidades
- **Cálculo em tempo real:** A Diária Média (ADR) atualiza instantaneamente à medida que os dados são inseridos.
- **Formatação Automática:** Todos os valores são processados com rigor e apresentados no formato da moeda Europeia (€).
- **Design Responsivo:** A interface adapta-se automaticamente a computadores e telemóveis.
- **Dark Mode:** Interface baseada nas cores oficiais do GitHub Dark Theme para maior conforto visual.

## 🛠️ Tecnologias Utilizadas
Este projeto foi desenvolvido de forma nativa e não requer instalações ou servidores:
- **HTML5** utilizado para Estrutura
- **CSS3** na Estilização com CSS Flexbox e Grid
- **JavaScript** como é obvio para a nossa lógica matemática

## 🚀 Como executar localmente
Como o projeto não tem dependências, basta:
1. Fazer o download do ficheiro `index.html`.
2. Abrir o ficheiro com qualquer navegador web.
